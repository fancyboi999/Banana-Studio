# 变更日志 (Changelog)

所有对本项目的重要更改都将记录在此文件中。

## [Unreleased]

### ✨ 新增
- **AI 图像生成**: 支持 Google Gemini 2.5 Flash 和 3.0 Pro 模型。
- **AI 图像编辑**: 支持基于自然语言的图像修改。
- **魔术分割**: 集成 Meta SAM-2 模型，实现一键对象分割与图层提取。
- **历史记录**: 本地 SQLite 数据库记录所有生成和编辑操作。
- **国际化**: 支持中英文界面切换。
- **编辑器 UI**: 全新的图层管理面板、属性控制面板和画布交互。
- **本地工具**: 裁剪、旋转、滤镜预设。

### 🛠️ 优化
- 升级至 Next.js 16 和 React 19。
- 迁移至 Tailwind CSS v4。
- 重构项目结构，采用 Server Actions 简化数据流。

### 🐛 修复
- 修复了若干 UI 布局问题。
- 解决了图层混合模式渲染不正确的问题。
